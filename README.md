# Classic_Cryptography
Python Scripts involving classical cryptography

### Methods such as:
- Caesar
- Progressive Caesar
- Quagmire III & IV
- Vigenere

These are mostly geared towards the Kryptos Sculpture challenge created by Jim Sanborn located at CIA Headquarters in Langley, VA.
